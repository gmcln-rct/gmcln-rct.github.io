PROJECTS

The Write Word

Web app that helps writers fiind word inspirations with synonyms, antonyms, rhymes, related words and more. Developed with React, react Query, SCSS and JavaScript.

seaOrganOf

seaOrganOf is inspired by the Sea Organ of Zadar creating an immersive audio experience using NOAA's Tides & Currents API data from stations around the U.S.

Swimeo

Swimeo is a full stack UI design exercise inspired by Vimeo --- with an aquatic twist! --- developed with React, Redux, AWS, Rails and CSS.

Brazilian Unit Converter

Converts between Brazilian recipe measurements and U.S. and metric units of volume, using React Hooks and Context API.

Honey Badger Institute of Higher Learning

Quiz app to help beginner programmeres to learn JavaScript, React. CSS and other frontend tools, through tough *Honey Badger Love*.

Wine Data Visualizations

An exploration of historical wine data using D3 that show trends in wine varietal production and consumption in various countries.

The SenseAbility Project

Interactive learning tool that shows everyday scenes as experienced by individuals with sensory processing disorder, or SPD, often associated with autism, using HTML5 and Bootstrap.

ephemeral Light

ephemeralLight is based on the tantric meditation method of Trāṭaka ("look" or "gaze"). The project uses the Tone.js JavaScript library to generate a tonal soundscape each time the button is pressed. Each individual soundscape -- and the flame audio-visualization it creates -- is unique and ephemeral.

The Wyr Deception

Immersive scifi narrative storyworld using video, graphic novels and short stories to build a narrative with a 1,000 year timeline.
